## ✨ Features

*   **Responsive Design:** Fully optimized for desktops, tablets, and mobile devices.
*   **Sticky Navigation:** A sleek header that transitions smoothly as the user scrolls.
*   **Dynamic Typing Effect:** Integrated `Typed.js` to showcase multiple titles dynamically on the hero section.
*   **Interactive Contact Form:** Built-in form allowing potential clients or employers to reach out.
*   **Social Integration:** Direct access to professional networks (LinkedIn, Facebook, Instagram, YouTube) and direct resume download links.

## 🛠️ Tech Stack Used

*   **Frontend:** HTML5, CSS3 (Custom styling and layout structure)
*   **Scripting:** Vanilla JavaScript
*   **Libraries:** 
    *   [Typed.js](https://github.com/mattboldt/typed.js/) (For the dynamic typing text effect)
    *   [Boxicons](https://boxicons.com/) (For clean, modern web icons)
